# F3 three-lift contribution instantiation v1

Status: `ONE_LAYER_OPERATOR_TO_PISTAR_PASS_COMPOSED_RENEWAL_OPEN`.

The one-layer three-channel stopped-path object is now consumed by the
row-wise operator contribution hook.  Retarded, advanced-direct and
parity-lift fibres are actual finite subsets of one parent `piStar` set, with
pairwise disjointness already audited.  The theorem therefore transports any
one-layer operator mass satisfying the explicit row-contribution inequality
to the parent `piStar` cardinality.

The contribution inequality itself is intentionally a premise.  This module
does not infer it from the frozen numerical matrix, does not compose layers,
and does not establish a renewal exponent or a density theorem.  Its purpose
is to make the remaining F3 semantic obligation executable: supply the
operator-to-contribution comparison for the composed paths.
